# Keen Keeper

Keen Keeper is a friendship and relationship tracking web app built with Next.js. It helps users keep track of their friends, monitor interaction frequency, review recent check-ins, and stay on top of important connections through a simple and visual interface.

## Live Demo

Live Link: [Click Here](https://keen-keeper-tau.vercel.app/)

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI
- Recharts
- React Icons
- React Toastify
- React Spinners

## Main Features

- Dashboard overview with quick stats such as total friends, on-track friends, and monthly interactions.
- Friends list with profile cards, tags, bios, contact details, and relationship status.
- Timeline page with search, filter, and sorting options for past check-ins.
- Analytics section with interactive charts to visualize interaction types.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser after starting the development server.
